Camera System Deployment Instructions

Deploy Cloudformation templates

  1. Enter CloudFormation in the search bar
  2. Ensure that the region (top right-hand corner) is set to 'United States (N. Virginia)
  3. Repeat the following steps first for the dashboard stack, then the cloudformation stack
    1. Select 'Create stack' from the top right-hand corner
    2. Select the 'With new resources (standard) option
    3. Select 'Upload a template file' and choose the CloudFrontStack.template.json file
    4. Click 'Next'
    5. Set the various CloudFormation Parameters
      • certificateArn should be the ARN of your certificate (copied earlier)
      • codeBucketName should be the bucket with the code in it - if you have changed from the default, alter it here
      • codeBucketRegion - the region the codeBucket is in
      • domainName - your domain name (excluding prefixes such as www. or auth.)
      • hostedZoneId - the Hosted Zone for your Route 53 domain
      • uniqueId - a unique number added to resources to prevent name conflicts, this could be the account number
    6. Click 'Next'
    7. On the next page most options can be left as default, the tick box for acknowledging CloudFormation might create IAM roles should be ticked
    8. Click 'Next'
    9. Click 'Submit' to create the stack
  4. Once the stack shows a status of CREATE_COMPLETE, select it
  5. Click on the 'Outputs' tab across the top
  6. Take note of the CloudFrontDistributionDomainName
