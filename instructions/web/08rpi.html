Camera System Deployment Instructions

Set up Raspberry Pi

  1. Download the Raspberry Pi Imager CloudFormation Raspberry Pi Software
  2. Put the SD card into the computer
  3. Run Raspberry Pi Imager
  4. Select the correct device
  5. Select 'Other OS', then 'Raspberry Pi Lite'
  6. Select the SD card as storage
  7. Click next
  8. Change the personalisation options
    • it is recommended you set up an SSH Key for access by clicking the button
    • ensure that the wireless network is configured
    • take note of the device's network name
  9. Click next
  10. Once the image is written to the SD card, swap card into the Raspberry Pi
  11. Switch the Raspberry Pi on
  12. Use scp to copy code files across to /home/user/app
  13. ssh into the Pi
  14. run pi_setup.sh
  15. run generate_config.sh and answer the questions
  16. reboot the Raspberry Pi
  17. Your system should now work!
