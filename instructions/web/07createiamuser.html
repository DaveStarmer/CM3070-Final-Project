Camera System Deployment Instructions

Create IAM User

  1. Search for 'IAM' in the AWS Search bar
  2. Select 'Users'
  3. Click create user
  4. Enter a user name and click next
  5. Select 'Attach Policies Directly'
  6. Click 'Create Policy'
  7. Select the JSON editor
  8. Paste in: { "Version": "2012-10-17", "Statement": [ { "Sid": "VisualEditor0", "Effect": "Allow", "Action": [ "s3:PutObject" ], "Resource": [ "arn:aws:s3:::vid-dash-upload-123456789012/*", "arn:aws:s3:::vid-dash-upload-123456789012" ] } ] } (replace 123456789012 with your unique number)
  9. View your new user
  10. Click 'Create Access Key'
  11. Select 'Other' then next
  12. Click 'Create Access Key'
  13. Take note of both your secret access key and your access key
